---- INSTRUCTIONS ---- 
Keys:
 - W: Move up
 - A: Move left
 - S: Move down
 - D: Move right
 - Left Click: Progress dialogue

Gameplay:
 - Collect gemstones and riddle pieces in order to survive the temple.
 - Multiple playable characters.

Cheat Codes:
 - C: Complete current puzzle (Player must be in the desired puzzle room.)

Final Puzzle:
 - Answer: Asha Ignatia
